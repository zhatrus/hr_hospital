# Product Context

## Проблема
Потрібен простий навчальний приклад Hospital Management для демонстрації моделей, в’юх та wizard-ів в Odoo.

## Цільова аудиторія
Розробники-початківці/студенти курсу Odoo.

## Основний потік
- Створення лікарів/пацієнтів
- Запис на візит
- Фіксація діагнозу/хвороби
- Допоміжні дії через wizard-и

## Унікальність
Концентрований приклад для навчання з кількома в’юхами і wizard-ами.
