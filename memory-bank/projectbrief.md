# Project Brief

## Основна мета
Навчальний модуль Odoo 17 для керування лікарнею: лікарі, пацієнти, візити, діагнози, розклади.

## Ключові вимоги
- [x] CRUD для Doctors/Patients/Visits/Diseases/Diagnoses
- [x] Зручні форми/дерева/меню
- [x] Wizard-операції (перепризначення лікаря, перенесення візиту тощо)

## Обмеження
- Odoo 17.0

## Цілі
- Стабільні та акуратні UI-в’юхи без “поламаних” блоків (особливо з `nolabel`).
